# Fashion Store Billing System

## 📌 Overview
A simple Fashion Store Billing System developed using HTML, CSS, and JavaScript. The application allows users to add products, calculate totals, generate bills, and print invoices.

## 🚀 Features
- Add customer details
- Add multiple dress items
- Select dress size
- Calculate total automatically
- Delete items
- Print bill
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
